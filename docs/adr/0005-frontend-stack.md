# Frontend stack: Vite + Tailwind v4 + TanStack Router

The SPA at `src/Balance.Web.Client` is Vite + React 19 + TypeScript with Tailwind v4 (CSS-first `@theme` configuration via `@tailwindcss/vite`, no `tailwind.config.js`), TanStack Router for file-based client routing (routes under `src/routes/`), `lucide-react` icons, self-hosted webfonts via `@fontsource/poppins` + `@fontsource/jetbrains-mono`, and Recharts for charts. All design tokens live in a single `@theme` block in `src/index.css`; components use Tailwind utilities directly (no CSS Modules). Component vocabulary matches `CONTEXT.md` — `JournalEntry` / `Counterparty` / `BankTransaction`, not everyday-speech equivalents.

Tailwind v4 is chosen because the design tokens are already CSS custom properties, so they serve as both the design source-of-truth and the Tailwind theme with no JS mirror. TanStack Router is chosen over React Router for compile-time type-safe routes and its pairing with TanStack Query. Recharts (over Visx, Nivo, Tremor, ECharts) composes charts as React JSX, accepts CSS-custom-property colour strings verbatim, and covers the v1 chart types (line/area/bar/pie).
